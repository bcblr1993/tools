# 软件资产扫描工具使用说明

本工具用于自动化收集 Windows/Linux/macOS 系统中安装的软件列表，以及 Python、Java、Node.js、Go、Rust 等语言的第三方插件包版本信息。

## 🛠 功能特点
- **零依赖**：不需要安装任何第三方 Python 库，直接运行。
- **全平台**：支持 Windows、Linux、macOS。
- **自动收集**：自动识别并收集系统软件及各类编程语言的依赖包。
- **Excel 友好**：生成的 CSV 报告支持 Excel 直接打开，中文不乱码。

## 🚀 如何使用

### 方式 1：Windows 环境
1. 进入文件夹。
2. **双击运行 `scan.bat`**。
3. 等待扫描完成，查看生成的 `reports` 文件夹。

### 方式 2：Linux / macOS 环境
1. 打开终端。
2. 进入脚本目录，执行：
   ```bash
   ./scan.sh
   ```
3. 扫描完成后，在 `reports` 目录下查看结果。

## 📂 结果说明
扫描完成后，会在 `reports` 目录下生成两个文件：
1. `[主机名]_系统软件_[时间].csv`：系统级别安装的软件（如 Chrome, Nginx, Office 等）。
2. `[主机名]_语言依赖包_[时间].csv`：开发语言相关的包（如 pip 安装的 requests, Maven 的 Spring 依赖等）。

---
*注：如果系统未安装 Python 3.6+，请先安装 Python 或联系开发人员获取打包好的 .exe 版本。*
